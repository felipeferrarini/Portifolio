## Getting Started

First, clone the repository and run this to install all packages:

```bash
npm install
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying the content in `pages/*`. The page auto-updates as you edit the file.
